In this figure, \(V_0\) is the effective forward velocity of the air over the blade. This is not exactly the forward velocity of the vehicle, \(V_{\inf}\) since there will be an induced component due to the generation of lift by the blade… \(V_2\) is the axial velocity of the propeller blade, which is a function of the blade element radius and the rotational speed.

We will define two adjustment variables, \(a\) and \(b\) that are used to calculate the actual velocities:

()¶\[\begin{equation} V_0 = V_{\infty} ( 1 + a) \end{equation}\]

If the rotational speed of the blade element is given by \(\Omega r\) Then

()¶\[\begin{equation} V_2 = \Omega r ( 1 - b) \end{equation}\]

From these two values, we can calculate the local flow velocity \(V_1\):

()¶\[\begin{equation} V_1 = \sqrt{V_0^2 + V_2^2} \end{equation}\]

The propeller is set at some pitch angle \(\theta\), which is measured from the plane of the propeller to the zero lift line of the blade airfoil section. The blade angle of attack is the angle between this flow velocity vector and the zero lift line for the blade airfoil section. From the blade section airfoil data, we will need to calculate the local lift and drag coefficients.

The angle between the zero lift line and the flow velocity vector is defined as:

()¶\[\begin{equation} \phi = \theta - \alpha = \tan^{-1}\left(\frac{V_0}{V_1}\right) \end{equation}\]

If we know \(V_0\) and \(V_1\) we can determine the angle of attack:

()¶\[\begin{equation} \alpha = \theta - \tan^{-1}\left(\frac{V_0}{V_1}\right) \end{equation}\]

The elemental thrust is defined by this equation:

()¶\[\begin{equation} \Delta T = \Delta L \cos{\phi} - \Delta D \sin{\phi} \end{equation}\]

The elemental torque is defined by this equation:

()¶\[\begin{equation} \Delta Q = r (\Delta L \sin{\phi} + \Delta D \cos{\phi}) \end{equation}\]

The local lift and drag forces can be obtained using the velocity \(V_1\) and the local lift and drag coefficients:

()¶\[\begin{equation} \Delta L = C_l \frac{1}{2}\rho V_1^2 c dR \end{equation}\]
()¶\[\begin{equation} \Delta L = C_d \frac{1}{2}\rho V_1^2 c dR \end{equation}\]

Conservation of Momentum¶

Now we apply the principle of conservation of momentum in both the axial and angular flow directions. For the moment, think of the propeller as a disk through which incoming air flows. The disk will acelerate the air to some new value with will then move beyond the propeller disc. (This view is called Actuator Disk Theory.)

The velocity of the air entering this stream tube is \(V_{\infty}\). The velocity exiting the stream tube far downstream is \(V_s\). The propeller will accelerate the flow as air passes through the disk.

The area of the propeller disk is:

()¶\[\begin{equation} A_{disk} = \pi R^2 \end{equation}\]

The angular velocity of the flow is:

()¶\[\begin{equation} \omega = 2\pi n \end{equation}\]

Where \(n\) is the propeller RPM.

The mass of air arriving at the disk is given by:

()¶\[\begin{equation} \rho * 2 \pi R dR \end{equation}\]
import math
import numpy as np

chord=0.10           #chord length of blade assumed constant with radius (meters)
pitch=1.0            #pitch distance in meters.
dia=1.6              #diameter of the propeller (meters)
R=dia/2.0            #tip radius 
RPM=2100.0           #engine speed in RPM
tonc=0.12*chord      #thickness to chord ratio for propeller section (constant with radius)
rho=1.225            #standard sea level atmosphere density (kg/m^3)
n=RPM/60.0           #RPM --> revs per sec
omega=n*2.0*math.pi  #rps --> rads per sec
B=2                  # 2 Bladed Propeller
print('omega',omega)

xs=0.1*R
xt=R
rstep=(xt-xs)/10
r1=np.linspace(xs,xt,10)

#calculate results for a range of velocities from 1 to 60 m/s
for V in range(1,61):
    #initialise sums thrust (N) torque (Nm)
    thrust=0.0
    torque=0.0
    # loop over each blade element
    for j in range(0,len(r1)):
        rad=r1[j]
        theta=math.atan(pitch/2/math.pi/rad)   # calculate local blade element setting angle
        sigma=2.0*chord/2.0/math.pi/rad        # calculate solidity    
        a=0.1                                  # guess initial values of inflow and swirl factor
        b=0.01
        finished=False                         # set logical variable to control iteration
        sum=1                                  # set iteration count and check flag
        itercheck=0
        #print("V",V,rad,theta,sigma)
        while not finished:
            V0=V*(1+a)                         # axial velocity
            V2=omega*rad*(1-b)                 # disk plane velocity
            phi=math.atan2(V0,V2)              # flow angle
            alpha=theta-phi                    # blade angle of attack
            cl=6.2*alpha                       # lift coefficient
            cd=0.008-0.003*cl+0.01*cl*cl       # drag coefficient
            Vlocal=math.sqrt(V0*V0+V2*V2)      # local velocity at blade
            DtDr=0.5*rho*Vlocal*Vlocal*B*chord*(cl*math.cos(phi)-cd*math.sin(phi))   # thrust grading    
            DqDr=0.5*rho*Vlocal*Vlocal*B*chord*rad*(cd*math.cos(phi)+cl*math.sin(phi))   # torque grading
            tem1=DtDr/(4.0*math.pi*rad*rho*V*V*(1+a))  # momentum check on inflow and swirl factors
            tem2=DqDr/(4.0*math.pi*rad*rad*rad*rho*V*(1+a)*omega)
            anew=0.5*(a+tem1)    # stabilise iteration
            bnew=0.5*(b+tem2)
 
            # check for convergence
            if abs(anew-a)<1.0e-5: 
                if abs(bnew-b)<1.0e-5:
                    finished=True
            a=anew
            b=bnew
            # increment iteration count
            sum=sum+1
            # check to see if iteration stuck
            if sum>500:
                finished=True
                itercheck=1
                print("convergence failed",V,rad,a,anew)
        thrust=thrust+DtDr*rstep
        torque=torque+DqDr*rstep
        # print("thrust",thrust,DtDr)
        # print("torque",torque,DqDr)
    t.append(thrust/(rho*n*n*dia**4))
    q.append(torque/(rho*n*n*dia**5))
    J.append(V/(n*dia))
    # print("A",V,t,q)
    eff.append(J[V-1]/2.0/math.pi*t[V-1]/q[V-1])
    icheck.append(itercheck)
Jmax = max(J)
Tmax = max(t)
print(Jmax, Tmax)
